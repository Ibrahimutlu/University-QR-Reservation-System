The Swagger UI generated by the backend at <http://localhost:5000/swagger>
is the live, machine-readable API contract.

---

## Regenerating Report 5

If you edit `generate-report5.py` (kept here for transparency), regenerate the PDF:

```bash
pip install reportlab
python docs/generate-report5.py
```

The output is written next to the script as `Report5-DatabaseAndIntegration.pdf`.
